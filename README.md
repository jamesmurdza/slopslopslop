# Hello World

A simple Hello World implementation in both Python and Rust.

## Overview

This repository contains minimal "Hello, World!" programs demonstrating basic syntax in two programming languages.

## Python Version

Run the Python implementation:

```bash
python hello.py
```

## Rust Version

Build and run the Rust implementation:

```bash
cargo run
```

Or build separately:

```bash
cargo build
cargo run
```

## Output

Both programs output:
```
Hello, World!
```

## Requirements

- Python 3.x (for hello.py)
- Rust toolchain with Cargo (for the Rust version)
